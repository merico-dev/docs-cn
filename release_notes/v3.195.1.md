# 版本更新说明 v3.195.1

| 版本号<br/>   | 更新日期<br/>   | 备注<br/>         |

## 功能优化

- EE-19746 最左侧导航栏（一级导航）支持拖拽右边缘调整宽度，以改变导航显示方式
- EE-19867 优化/新增“专属体检报告”看板，辅助 2026 devdata 调研汇报
- EE-19891 优化系统默认报表页面数据说明的展示
- EE-19894 优化用户资料页数据评估项目名称的显示，增加可理解性
- EE-19895 优化成员页 VP 列表显示逻辑，支持展示未设置成员的 VP
- EE-19931 代码库分析列表 UI 显示问题，包括对齐、图标显示和数字格式优化
- EE-19995 Line Diff 方式支持.cptx 后缀文件计算当量（.cptx 本质是 XML 文件）
- EE-20008 分析导出数据起始时间调整为 2024-01-01

## 新增功能

- EE-19648 重构成员个人页：补齐个人基础数据的显示；补齐个人在所有领域的数据；取消统计数据上的 viewer 权限限制，限制控制在明细数据上
- EE-19916 devtable 树状多选筛选器增加选项浏览功能。
- EE-19950 雷达图支持 series 样式自定义（颜色、线宽、背景透明度）

## 指标数据变化

- AE 使用 Adjustments 替换 Files 表

  - commit 详情的文件列表，以及/repo/commit/get-punish 接口会收到影响，需要重新分析
  - commit 列表页的当量调节信息，不受影响
- 个人页上线，综合质量评价统计口径变成最近一年的新增函数与代码问题，原来质量数据是切面的全部
- 基线有调整，原来是自己算的基线（实时根据用户数据权限范围内的数据计算），现在是企业基线（devinsight 使用系统里的数据定期计算）
- 虚拟项目原来不指定成员会包含没有在系统创建的账号（贡献者），现在只包含 License 里、已启用的账号
- 虚拟项目_/openapi/virtual_project/add, _当不指定 accounts 时,必须要传参数_"autoSyncAccount": true_
