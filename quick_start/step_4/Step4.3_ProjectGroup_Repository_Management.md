# 项目组代码库管理

<br>

## 注意事项

- **操作需要权限：** 操作账户有以下两种权限皆可：
    1. 思码逸系统 **超级管理员** 权限。 
    2. 有进入 **项目设置** 界面的角色权限，并拥有此项目组或更高层级项目组的访问权限。

- 可进行操作：新增代码库 / 删除代码库 / 移动代码库。

<br>

## 设置导航

菜单 **设置** -> 左侧菜单栏 **项目设置**
