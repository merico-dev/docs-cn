# 开发负载

<br />

## 要点概述

-   **导航：** 左侧菜单栏 **项目表现 - 开发负载**。

-   在此界面展示了人均生产率趋势、年度同期生产率均值之比（排除异常点）、贡献者帕累托、开发活动热力图，用以判断人均开发工作量的负荷程度，贡献是否均衡，以及判断资源的利用是否合理。

<br />
